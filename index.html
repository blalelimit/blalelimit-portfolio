PNG Images to ICO

Converting one image format to another has never been so easy with FOSS software. Just always read the documentation or manual. {% highlight python %} # Simple Way to Convert *.png files to *.ico using PIL from PIL import Image img = Image.open(r'inputs/input.png') img = img.resize(size=(64, 64), box=None, reducing_gap=None, resample=Image.Resampling.LANCZOS) img.save('outputs/output.ico', 'ico', sizes=[(64, 64)]) {% endhighlight %} {% highlight bash %} # ImageMagick can also do the same thing convert input.png -resize 64 output.ico {% endhighlight %}
